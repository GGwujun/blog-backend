# 苹果电脑 macOS 系统打开终端的方法：

点击 "Launchpad"-"其他"-"终端"。

# Windows 系统打开终端的方法：

按住 "Win" 键（一般在 Alt 键的左侧）的同时按一下 "R" 字母键，在弹出的窗口中输入 "powershell" 然后回车。
